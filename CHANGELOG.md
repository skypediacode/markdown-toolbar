# Changelog

## v1.0.0

- Markdown formatting toolbar in the editor title bar.
- Right-click context menu with `Bold`, `Italic`, `Link`, and a `More` submenu.
- `More` submenu includes headings, lists, math, tables, code blocks, and utilities.
- Keyboard shortcuts for all formatting actions (`Ctrl+B`, `Ctrl+I`, `Ctrl+K`, and `Ctrl+M` chords).
- Toggle-aware inline formatting (bold, italic, strikethrough, inline code, inline math).
- Line-based transforms for headings, quotes, and lists.
- `Update Section Numbers` to renumber H1–H3 headings for a selection or the whole file.
- `Update List Numbers` to renumber ordered lists for a selection or the whole file.
